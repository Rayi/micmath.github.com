JSDoc 3 Tag Dictionary

@augments or @extends
This object adds onto a parent object.
@borrows
This object uses something from another object.
@constructor or @class
This function is intended to be called with the "new" keyword.
@classdesc
Use the following text to describe the entire class.
@constant or @const
Document an object as a constant.
@copyright
Document some copyright information.
@constructs
This function member will be the constructor for the previous class.
@defaultvalue or @default
Document the default value of a variable.
@deprecated
This is no longer the preferred way.
@desc or @description
Describe the object.
@event
Document an event.
@example
Provide an example.
@throws or @exception
Describe what errors could be thrown.
@exports
Document the name of a JavaScript module.
@file or @overview or @fileoverview
Describe a file.
@fires
Describe the events this object may fire.
@function or @method
Describe a function.
@global
Document a global object.
@ignore
Remove this from the final output.
@inner
Document an inner object.
@instance
Document an instance member.
@kind
What kind of object is this?
@license
Document the software license that applies to this code.
@memberof or @member
This object is a member or another object.
@mixes
This object mixes in all the members from another object.
@mixin
Document a mix-in object.
@module
Document a JavaScript module.
@name
Document the name of an object.
@namespace
Document a namespace object.
@param or @arg or @argument
Document the parameter to a function.
@private
This member is meant to be private.
@property
Document a property of an object.
@protected
This member is meant to be protected.
@public
This member is meant to be public.
@readonly
This variable is meant to be readonly.
@requires
This JavaScript module requires another JavaScript module.
@returns or @return
Document the return value of a function.
@see
Refer to some other documentation for more information.
@since
When was this feature added?
@summary
A shorter version of the full description.
@this
What does the this keyword refer to here?
@type
Document the type of an object.
@typedef
Document a custom type.
@version
Document what version this implementation is.
